# 📝 Formulaire d'Inscription

Un formulaire moderne et responsive développé en **HTML5** et **CSS3**.

![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3)
![Responsive](https://img.shields.io/badge/Responsive-Yes-success)

---

## ✨ Fonctionnalités

- ✅ Validation HTML5 (email, téléphone, mot de passe)
- 📱 Design responsive (mobile & desktop)
- ♿ Accessibilité (labels, messages d'aide)
- 🎨 Animations CSS au focus et au survol
- 🔒 Confirmation de mot de passe

---

## 🚀 Utilisation

### 1. Cloner le projet
```bash
git clone https://github.com/elhaik-amine/formulaire-inscription.git
cd formulaire-inscription
```

### 2. Ouvrir dans le navigateur
#### Windows :
```bash
start formulaire.html
```
#### macOS/Linux :
```bash
open formulaire.html
```
